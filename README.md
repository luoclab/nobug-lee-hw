# nobug-lee-hw

李宏毅老师的课程已经成为了AI入门学习者的最火课程之一，在中国有众多的学习者

但是绝大多数学习者只限定于学习内容本身，忽略了学习课程之外的和课程配套的**课程作业**

课程作业里面包含了绝大多数老师课堂上的内容，如果在学习课程后及时的完成和课程配套的代码作业对于学习者来说是非常有帮助的

但是在作者完成配套作业的课程中遇到了大量的代码问题，包括环境错误，代码错误，无法运行等等

在这样的背景下，作者希望开发出完全没有环境和版本问题的作业，**让想要完成课程作业的同学可以只专注于课程内容本身，而无需关注任何的环境问题**

## 项目受众

本项目适合以下人群学习：

🎓 **看过李宏毅老师的课程，希望理论学习以外也可以动手实操的同学**

💻 希望可以快速进行**动手实践深度学习**的同学

🤔 任何想要**无脑**体验深度学习的同学（**只需点击运行按钮**）

## ✨ 项目亮点

### 对于零基础的同学

1.  **无脑运行**：对于完全零基础的同学也可以跑通所有代码，只需点击运行就可以
2.  **不需要管理环境**：没有bug,没有环境和安装包问题，学习者只需要专注于学习内容本身
3.  **在线编程**：只在colab上进行编程，只要能打开colab，就可以跑完所有代码
